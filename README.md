# 畅聊系统(UdpChatSystem)

###  1.项目的基本原理

* 基于Udp的群聊系统
* 服务器基于生产者-消费者模型，一个线程收到客户端的数据，该数据放入数据池，另一个线程负责从数据池中拿到数据群发给在线列表中的所有好友

* 基于环形队列实现数据池

* 借助ncurses库来模拟界面

* 借助jsoncpp实现数据的序列化与反序列化 

##  2.聊天室功能

* 客户端可以发送消息给服务器
* 服务器向所有在线列表中的好友转发数据
* 客户端退出，其他客户端的在线列表中就没有该好友

###  3.项目缺点

* UDP丢包问题
* 如果是quit报文丢失了怎么办
* 性能问题，数据池被打满
* 可以加入超时重传(如果接受缓冲区打满会出现很大的问题)

### 4. 项目扩展

* 管理客户(超时间断开连接)
* 模拟注册功能
* 模拟登录过程
* 基于ncurses库实现点击在线列表中的好友进行单聊
